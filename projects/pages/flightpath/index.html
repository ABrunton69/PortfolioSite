Flight Path Generator App

A shortest path generator for small planes using C# and Windows Forms

Project Overview

The Flight Path Generator App is a Windows Forms application written in C# that helps small aircraft pilots and planners determine the shortest and most efficient flight paths between airports. The app uses distance between coordinates to calculate optimal routes, taking into account various constraints and user inputs.

  • Built with C# and Windows Forms
  • Created for my A-Level Computer Science NEA
  • User-friendly interface for entering airports and waypoints
  • Visualizes the calculated flight path on a map

Key Features

  • Interactive user input for selecting start and end points, fuel quantity and plane size
  • Save flight plan as CSV
  • Responsive UI
