Project Overview

N8N is an open source AI workflow automation tool that I host on my debian linux server. It allows me to create the following demonstrations that remove manual tasks I have to do in day to day life

  • Github tracker that tracks when I or another use pushes to my git account, records details of the push and posts it to a discord channel
  • RSS Tracker that tracks https://krebsonsecurity.com/ & https://thehackernews.com/ for new articles, it gives a brief overview of the article and stores it in a google sheet after sending it to discord
