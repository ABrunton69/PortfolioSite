Itinerary Manager App

A Itinerary manager for a company who plans and sets activities using Java

Project Overview

This project was built for my Object Oriented Programming module in the first year of University, it takes into account the user of objects for the itinerary and the activities and uses classes to perform calculations, saving details to text files and the graphical interface of the application.

  • Built with Java
  • Implements Swing (A graphical user interface library)
  • User-friendly interface for reading itinerary data

Key Features

  • Performs numerical calculations on currency values
  • Exports itinerary data to text files to store data
  • A command line interface for entering new activities and itinerary's
